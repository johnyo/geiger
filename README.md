geiger
======

Low Voltage 2.5V PIN Diode Based Geiger Counter

Purpose
======

Super low voltgae Geiger Counter using a modified PIN diode. This is a complete Geiger Counter project modified from the PIN diode suggestion of Bernd Laquai here: http://www.opengeiger.de/ABGDetektor_en.pdf

Project Structure
=================

/GeigerCounter
* The Eagle PCB project

/FabricationOutput
* This holds the fabrication output script for the 2 layer board
